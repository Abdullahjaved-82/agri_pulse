1. Introduction

Agriculture is a major part of Pakistan's economy, contributing approximately 19.2% to the GDP and employing around 38.5% of the country's labor force. Many farmers rely on local markets (mandis) to sell their crops. However, farmers often lack easy access to updated crop prices and market trends. As a result, they sometimes sell crops at lower prices due to limited information.

With the growth of smartphones across rural and semi-urban areas of Pakistan, a mobile application can help farmers and traders easily access crop price information, market trends, and useful agricultural tools. The proposed application AgriPulse aims to provide a simple yet powerful platform where users can check mandi prices, monitor trends, and access helpful farming information — all from the convenience of their mobile phones.

This document presents the mid-term progress of the AgriPulse project, focusing on the complete User Interface (UI) implementation built using the Flutter framework. The document includes detailed descriptions and screenshots of every screen developed so far.

2. Problem Statement

Farmers and traders in Pakistan often face significant difficulties in obtaining updated crop price information from different mandis (agricultural markets). Most farmers depend on middlemen or physical visits to markets to know current prices. This process is inefficient, time-consuming, and sometimes leads to unfair pricing due to information asymmetry.

The lack of a centralized, digital platform for accessing real-time mandi data means that farmers are frequently at a disadvantage when negotiating crop prices. There is a critical need for a mobile application that provides easy, transparent access to crop prices and agricultural market intelligence to help farmers make better, data-driven selling decisions.

3. Proposed Solution

The proposed solution is a mobile application called AgriPulse, which provides crop price information from different markets along with useful farming tools and market insights. The application offers an easy-to-use, visually appealing interface where users can:

  • View current crop prices from various mandis across Pakistan
  • Monitor historical price trends using interactive charts
  • Locate nearby mandis and view their details
  • Set price alerts for specific crops
  • Calculate profit using a built-in profit calculator
  • Stay updated with agriculture news
  • Check weather advisories relevant to farming

For the mid-term submission, the focus has been on developing a complete, polished user interface (UI) with sample/dummy data. Backend services and real-time data integration will be added in subsequent phases.

4. Objectives

The objectives of the AgriPulse project are:

  • To develop a mobile application that displays crop market prices from mandis across Pakistan.
  • To design a user-friendly, visually appealing interface suitable for farmers and traders of all technology skill levels.
  • To provide crop price analysis and market insights through interactive charts and analytics.
  • To introduce helpful tools for agricultural decision making, including a profit calculator and price alerts.
  • To build a scalable mobile application architecture that supports future enhancements such as AI-based predictions and real-time data.
  • To implement a complete authentication flow with login, signup, password recovery, and OTP verification screens.

10. Future Enhancements

The AgriPulse application is designed with scalability in mind. The following enhancements are planned for future development phases:

  • Firebase Integration: Backend database and authentication services using Firebase for real user accounts and data storage.
  • Real-Time Price Data: Integration with government mandi price databases and web scraping for live crop prices.
  • AI-Based Price Prediction: Machine learning models to forecast crop prices and help farmers plan better.
  • Google Maps Integration: Interactive maps showing mandi locations with directions and distance calculations.
  • Farmer-to-Buyer Marketplace: Direct trading platform connecting farmers with buyers to eliminate middlemen.
  • Weather API Integration: Real-time weather data from APIs for accurate forecasts and irrigation advisories.
  • Multi-Language Support: Complete Urdu translation for all app screens to serve a wider user base.
  • Push Notifications: Cloud messaging for real-time price alerts and important market updates.
  • Data Analytics Dashboard: Advanced charts and export capabilities for detailed market analysis.
  • Offline Support: Cached data and offline mode for areas with limited internet connectivity.

The AgriPulse mobile application aims to improve access to agricultural market information for farmers and traders across Pakistan. This mid-term submission demonstrates the complete User Interface (UI) implementation of the application, built using the Flutter framework with 20 fully functional screens covering authentication, market data display, analytical tools, news, and user profile management.

The application features a clean, modern design with a consistent green color theme representing agriculture, smooth navigation flows, interactive charts for data visualization, and practical tools like the Profit Calculator and Price Alert system. The use of Flutter ensures cross-platform compatibility with both Android and iOS devices from a single codebase.

By providing mandi price data, market analytics, and helpful farming tools through an intuitive mobile platform, AgriPulse can support better decision-making and increase transparency in crop markets. This project demonstrates how mobile technology can effectively address real-world agricultural challenges and empower Pakistan's farming community with accessible, actionable market intelligence.
